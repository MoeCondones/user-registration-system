Maven Dependencies

This project uses Maven for dependency management, which automatically downloads the required libraries.

The main dependencies are:

1. JUnit 5 (Jupiter) - Latest version 5.10.1:
   - junit-jupiter-api
   - junit-jupiter-engine

2. Mockito - Latest version 5.8.0:
   - mockito-core
   - mockito-junit-jupiter

If you're not using Maven, you can download these libraries from Maven Central Repository:
https://mvnrepository.com/

To run the project with Maven:
```
mvn clean test
```
